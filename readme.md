# pacman-diffs

![](http://xn--pp-oia.com/dl/Hgq8)

## Installation
With AUR :

    yaourt -S pacman-diffs
    
## Usage

	 -x extension (pacnew, pacsave...)
	 -d since x days
	 -l : list only - no diff
	 -e filename : edit files
	 -a archive pacnew in /var/cache/pacman/pacnew/
	 -c compare archives
	 -i pacman history
	 -j pacman history files
